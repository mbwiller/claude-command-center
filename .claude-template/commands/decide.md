# Decide Command

Analyze complex decisions from multiple perspectives using the @consensus agent.

## Usage
```
/decide [question or decision to make]
```

## Process
1. Spawn @consensus agent
2. Frame the decision clearly
3. Analyze from perspectives:
   - Developer Experience
   - Architecture
   - User Impact (HackLearn Pro learners)
   - Resource Cost
   - Security
   - Time Horizon
4. Create perspective matrix
5. Synthesize recommendation
6. Document rationale
7. Update @memory-keeper with decision

## Arguments
$ARGUMENTS - The decision or question requiring multi-perspective analysis

## Examples
```
/decide should we use WebSocket or SSE for the LLM playground streaming?
/decide React Query vs SWR for data fetching
/decide monorepo vs polyrepo for HackLearn Pro modules
```

## Output Format
- Options considered with pros/cons
- Perspective matrix (5-star ratings)
- Clear recommendation
- Confidence level (HIGH/MEDIUM/LOW)
- Dissenting view acknowledgment
